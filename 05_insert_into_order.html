Insert Into

Definition

The INSERT INTO statement in SQL is used to add new rows of data into a table.

Code

INSERT INTO
        orders (user_id, product, amount)
    VALUES
        (1, 'Laptop', 1200.00),
        (2, 'Tablet', 250.00),
        (12, 'Phone', 800.00),
        (15, 'Monitor', 300.00),
        (7, 'Keyboard', 100.00),
        (4, 'Mouse', 50.00),
        (3, 'Headphones', 150.00),
        (9, 'Printer', 350.00);
